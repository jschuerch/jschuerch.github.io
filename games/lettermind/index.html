Lettermind

Guess the word

A small deduction game inspired by Wordle and Mastermind.

Guess the hidden word. After each guess:

  • Green shows how many letters are in the correct position
  • Yellow shows how many letters exist in the word, but are misplaced

Use the marker tools to keep track of clues, ideas, and eliminated letters while solving.

  • Markers can be applied to both guessed tiles and keyboard letters
  • The red marker automatically propagates to all matching letters and is useful for marking letters that are definitely not in the word
  • Hold Shift while marking to propagate any marker color to all matching letters
